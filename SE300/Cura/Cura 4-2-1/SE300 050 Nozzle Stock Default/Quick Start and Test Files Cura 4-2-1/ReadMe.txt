Open these Cura 4.2.1 Project files to

> create a new dual machine profile for 0.5mm filament
> import the STL dual color model
> create a new slicing profile for 0.5mm nozzle


1. Choose File> "Open File(s)..." in Cura 4.2.1
2. Choose "SeeMeCNC_Dual_Batarang........." then click OPEN
3. Select "Open as project"
4. Select "Open"
5. Click "Slice" in the lower right corner of Cura 4.2.1
6. Then click "Save to File"


Upload to your SeeMeCNC 3D printer and run.
