Quick Start to printing as of July 2019. by Steve Wygant
2019 - SeeMeCNC

The .curaproject file is saved from a Cura 3.4.1 installation. 

Opening a .curaproject file in Cura 3.4.1 will automatically

1. Create a BOSSdelta machine profile
2. Create / Load a test STL
3. Create / Update a slicing profile

Notes:
Open as a project and overwrite all previous data to 
  reset or initially install.
